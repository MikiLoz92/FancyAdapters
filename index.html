What is FancyAdapters?

FancyAdapters is a collection of customizable RecyclerView Adapters for Android, that provide item selection, contextual action mode controls, drag&drop and swiping, among other functionalities.

This is the way your users should be allowed to treat their own data: intuitively, easily and, above all, on a functional basis.

Installation

In your project's build.gradle add jitpack's repository:

allprojects {
    repositories {
        ...
        maven { url "https://jitpack.io" }
    }
}

And in your module's build.gradle add the following dependency:

dependencies {
    ...
    compile 'com.github.mikiloz92:fancyadapters:0.3.1'
}
